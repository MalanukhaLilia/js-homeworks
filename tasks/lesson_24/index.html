24. JavaScript Basics

All possible use cases for JS:

  • Data storage and auto-saving
  • Responding to user actions (e.g., clicks and text input)
  • Validation
  • Requesting permission to use the microphone, camera, or geolocation...
  • Creating menus (like a burger menu) more conveniently than with pure CSS
  • Calculators and timers
  • Filtering items
  • Integrations (e.g., Google Maps)
  • Lazy loading/infinite scroll for items
  • Regular expressions (Regex)
  • Drag-and-drop file uploads
  • Pop-up tooltips triggered by user actions
  • Image processing (e.g., image compression)
  • Hotkeys / keyboard shortcuts
  • 3D visualization of models
  • Changing the page language
  • Dark mode
